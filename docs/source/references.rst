How to cite Duo
***************

The main reference is [Duo]_.

For the exponential decay expansion ``POLYNOM_DECAY_24`` cite [17PrJaLo]_.

For the continuum (``unbound``) states identification (``thresh_bound``) and unbound/bound line lists processing cite [22YuNoAz]_.

For the Omega representation cite [26BrYu]_.

For the using the hyperfine solver in Duo cite [22QuYuMe]_. 

For electric quadrupole calculations cite [21SoYuYa]_ and [24SoYuKi]_. 

Foe magnetic dipole calculations cite [24SoYuKi]_.
