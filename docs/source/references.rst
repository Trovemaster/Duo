How to cite Duo
***************

This manual uses BibTeX-based citations via the Sphinx extension
``sphinxcontrib-bibtex``.

Use the ``:cite:p:`` role for parenthetical citations and ``:cite:t:`` for
textual citations.

Examples
========

* Main program reference: :cite:p:`Duo`.

* Exponential decay expansion ``POLYNOM_DECAY_24``: :cite:p:`17PrJaLo`.

* Continuum (``unbound``) state identification and bound/unbound line-list
  processing (``thresh_bound``): :cite:p:`22YuNoAz`.

* :math:`\Omega` representation: :cite:p:`26BrYu`.

* Hyperfine solver in Duo: :cite:p:`22QuYuMe`.

* Electric quadrupole calculations: :cite:p:`21SoYuYa,24SoYuKi`.

* Magnetic dipole calculations: :cite:p:`24SoYuKi`.

* Lande g-factors: :cite:p:`16SeYuTe`.
